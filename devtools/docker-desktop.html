2. 配置镜像加速 ​

配置Docker Engine，添加docker加速镜像如下：

sh
{
  "builder": {
    "gc": {
      "defaultKeepStorage": "20GB",
      "enabled": true
    }
  },
  "experimental": false,
  "registry-mirrors": [
    "http://hub-mirror.c.163.com"
  ]
}

3. docker-compose安装 ​

3.1. 查看版本

sh
docker-compose version
1

默认docker-desktop会集成安装，如果未安装则没有版本信息

3.2. 安装brew

  1. 中科大镜像安装
sh
/bin/bash -c "$(curl -fsSL <https://gitee.com/cunkai/HomebrewCN/raw/master/Homebrew.sh>)"
1
  1. 更换下载源
sh
git -C "$(brew --repo)" remote set-url origin git://mirrors.ustc.edu.cn/brew.git
1

更换brew

sh
git -C "$(brew --repo homebrew/core)" remote set-url origin git://mirrors.ustc.edu.cn/homebrew-core.git
1

更换homebrew-core

sh
git -C "$(brew --repo homebrew/cask)" remote set-url origin <https://mirrors.ustc.edu.cn/homebrew-cask.git>
1

更换homebrew-cask

sh
echo 'export HOMEBREW_BOTTLE_DOMAIN=<https://mirrors.ustc.edu.cn/homebrew-bottles>' >> ~/.bash_profile
1

更换homebrew-bottles

sh
source ~/.bash_profile
1

让bottles配置生效

  1. 更新软件包信息
sh
brew update
1

参考文章:

https://zhuanlan.zhihu.com/p/665201467

3.3. 安装docker-compose

sh
brew install docker-compose
1
